# Hexadecimle
**Hexadecimle** is a game I made, inspired by the [Wordle](https://www.nytimes.com/games/wordle/index.html), but instead of guessing words, you have to accurately guess the `hex code` color of the background. Input the color as a hex code in the RGB boxes and press `submit guess` to see how close your guess was to the actual color. The area in the center of the window displays the color you guessed for visual comparison to the background color. See how many guesses it takes for you to get the color and solve the Hexadecimle!
